# Boxers Bed & Biscuits

Website for Boxers Bed & Biscuits, a full-service pet care campus in Belpre, OH featuring daycare, boarding, grooming, enrichment, veterinary care, and training — all under one roof. Part of the [Embark Pet Services](https://www.embarkpetservices.com/) portfolio operated by Cadence Private Capital.

**Current site (being replaced):** https://www.boxersbedandbiscuits.com/
**Sister site (design & technical reference):** https://houndaroundresort.com/

- **Dev server**: `npm run dev` (runs on http://localhost:3000)

- **Production server**: `npm run start`
- **Lint**: `npm run lint`
- **Sanity Studio**: `npm run sanity:dev` (runs on http://localhost:3333)
