# Boxers Bed & Biscuits

Website for **Boxers Bed & Biscuits**, a full-service pet care campus in Belpre, OH. Built with Next.js 16, Sanity CMS, and Tailwind CSS v4.

**Live site:** https://boxersbedandbiscuits.com/

## Three Sub-Brands

1. **Boxers Bed & Biscuits (PAW-PLEX)** — Daycare, boarding, grooming, training
2. **Boxers Enrichment Center (BEC)** — Private play and guided enrichment
3. **Boxers Meds & Fixits** — Veterinary clinic

## Getting Started

```bash
npm install
cp frontend/.env.example frontend/.env.local
# Add your Sanity project ID and read token
npm run dev
```

- Frontend: http://localhost:3000
- Sanity Studio: http://localhost:3333

## Tech Stack

- **Framework:** Next.js 16 (App Router, React 19)
- **CMS:** Sanity v5
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Hosting:** Vercel
- **Fonts:** Playfair Display + Rubik
